Notations

R1,R2 - General Purpose Registers A,B,C,D,E,H,L
M - Memory Location pointed by HL register pair
Addr. - 16 bit Address
Data - 16 bit Data
Rp - Register Pairs - H(HL), B(BC), D(DE)
Port - 8 bit port address
  • MOV
    Formates:
    1. MOV R1,R2 - The contents of R2 is moved into register R1
    example:-
    MOV B, E : Transfers data from Register E to B. Here Transfers means copying, not cutting.
    • B E
    Addr. Mode Flags Affected Cycles T-States
    Register None 1 4
    2. MOV R1, M - Content of Memory Location pointed by Register pair HL is transferred to register R1
    example: -
    MOV B, M // assuming content in HL is 2010H this instructions transfers content of memory location pointed by Register pair HL. HL have 2010, so content of memory location 2010H will be transferred to register B
    • B[[ HL]]
    • B[ 2010H]
    Addr. Mode Flags Affected Cycles T-States
    Indirect None 2 7
    3. MOV M, R2 - Content of Register R2 is transferred to memory location pointed by register pair HL
    example: -
    MOV M, B // assuming content in HL is 2010H, this instruction transfers content of register B to memory location pointed by HL register pair. HL contains 2010, so content of register B will be transferred to memory location 2010H
    • [[ HL]] B
    • [ 2010H] B
    Addr. Mode Flags Affected Cycles T-States
    Register None 2 7
