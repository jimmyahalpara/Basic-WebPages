Notations

R1,R2 - General Purpose Registers A,B,C,D,E,H,L
M - Memory Location pointed by HL register pair
Addr. - 16 bit Address
Data-16 - 16 bit Data
Data-8 - 8 bit data
Rp - Register Pairs - H(HL), B(BC), D(DE)
Port - 8 bit port address

Arithmetic Instruction set

  • MOV
    Forms:
    1. MOV R1, R2 - The contents of R2 is moved into register R1
    example:-
    MOV B, E : Transfers data from Register E to B. Here Transfers means copying, not cutting.
    • B E
    Addr. Mode Flags Affected Cycles T-States Bytes
    Register None 1 4 1
    2. MOV R1, M - Content of Memory Location pointed by Register pair HL is transfered into register R1
    example: -
    MOV B, M // assuming content in HL is 2010H this instructions transfers content of memory location pointed by Register pair HL. HL have 2010H, so content of memory location 2010H will be transferred into register B
    • B [[ HL ]]
    • B[ 2010H ]
    Addr. Mode Flags Affected Cycles T-States Bytes
    Indirect None 2 7 1
    3. MOV M, R2 - Content of Register R2 is transferred into memory location pointed by register pair HL
    example: -
    MOV M, B // assuming content in HL is 2010H, this instruction transfers content of register B into memory location pointed by HL register pair. HL contains 2010, so content of register B will be transferred into memory location 2010H
    • [[ HL ]] B
    • [ 2010H ] B
    Addr. Mode Flags Affected Cycles T-States Bytes
    Register None 2 7 1
  • MVI
    Forms:
    1. MVI R1, data-8 - The 8 - bit data is immidiately moved into register R1. Example:
    MVI B , 20H - data 20H will be transferred into register B
    • B 20H
    Addr. Mode Flags Affected Cycles T-States Bytes
    Immidiate None 2 7 2
    2. MVI M, data-8 - 8-bit data immidiately moved into the memory location pointed by HL (memory pointer)
    Example:
    MVI M, 20H //assuming data in HL pair is 2010H, this instruction will transfer data 20H to memory location pointed by HL. HL register pair have data 2010H, so 20H will be transfered to memory location 2010H
    • M 20H
    • [[ HL ]] 20H
    • [ 2018H ] 20H
    Addr. Mode Flags Affected Cycles T-States Bytes
    Immidiate None 3 10 2
  • LXI
    Forms: 1. LXI Rp, data-16 - The 16 - bit data is immidiately moved into the Rp register pair specified in the instruction. Registor pairs can be HL - H, BC - B, DE - D Example:
    LXI D, 2910H - data 2910H is immidiately moved to DE register pair.
    • HL 2910H
    Addr. Mode Flags Affected Cycles T-States Bytes
    Immidiate None 3 10 3
  • LDA
    Forms 1. LDA Addr. - The accumulator is loaded with the contents from the memory location having given address
    Example:
    LDA 2019H - content from memory location 2019H is loaded into accumulator.
    • A [ 2019H ]
    Addr. Mode Flags Affected Cycles T-States Bytes
    Direct None 4 13 3
  • STA
    Forms 1. STA Addr. - Contents from accumulator is stored into momory with location address
    Example:
    STA 2019H - content from accumulator is stored in memory with location 2019H
    • [ 2019H ] A
    Addr. Mode Flags Affected Cycles T-States Bytes
    Direct None 4 13 3
  • LHLD
    Forms.
    1. LHLD Addr. - The HL pari is loaded with the contents of the locations pointed by the given addr and addr. + 1
    Example:
    LHLD 2019H - contents of location pointed by address 2019H will be transfered to register L and contents of location pointed by address 2020H (2019H + 1) will be transferred to register H
    • L [ 2019H ] & H [ 2020H ]
    Addr. Mode Flags Affected Cycles T-States Bytes
    Direct None 5 16 3
  • SHLD
    Forms.
    1. LHLD Addr. - The content of HL pari is transferred to memory locations pointed by the given addr and addr. + 1
    Example:
    SHLD 2019H - contents of L will be transferred to memory location pointed by address 2019H and contents of H will be transferred to memory location pointed by address 2020H
    • [ 2019H ] L & [ 2020H ] H
    Addr. Mode Flags Affected Cycles T-States Bytes
    Direct None 5 16 3
  • LDAX
    Forms: 1. LDAX Rp - The accumulator is loaded with the memory location pointed by value of the given Register pair
    Example
    LDAX B // assuming data in BC register pair is 1010H. content from memory location 1010H will be transferred into accumulator
    • A [[ BC ]]
    • A [ 1010H ]
    Note: Register pair here cannot be HL
